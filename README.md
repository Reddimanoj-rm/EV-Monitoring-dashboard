# 🚗 EV Monitoring Dashboard in Microsoft Excel

An interactive **Electric Vehicle (EV) Monitoring Dashboard** built using **Microsoft Excel** to analyze EV performance, battery health, charging behavior, maintenance status, and operational metrics. This project demonstrates advanced Excel dashboarding techniques using Pivot Tables, Pivot Charts, Slicers, KPIs, and Conditional Formatting.

---

## 📌 Project Overview

The EV Monitoring Dashboard provides a comprehensive view of electric vehicle operations by transforming raw data into meaningful insights. It enables users to monitor battery performance, charging efficiency, energy consumption, maintenance status, and vehicle utilization through an intuitive and interactive dashboard.

This project is ideal for showcasing **Data Analytics**, **Business Intelligence**, and **Excel Dashboard Development** skills.

## ▶️ How to Use

1. Download or clone this repository.
2. Open **EV_Monitoring_Project.xlsx** in Microsoft Excel (2019/2021 or Microsoft 365).
3. Go to the **Dashboard** worksheet.
4. Use the slicers to filter the data.
5. Explore the KPIs and charts for insights.
